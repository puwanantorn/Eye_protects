# 👁️ Eye Protects — AI Eye Disease Screening Application

Eye Protects is an AI-powered mobile healthcare prototype designed to support early screening of eye diseases using:

- 📷 Eye image analysis
- 💬 Symptom-based diagnosis

The application aims to improve accessibility to preliminary eye health assessment, particularly for:

- Elderly users
- Remote communities
- Medical screening environments
- Educational and research use

⚠️ This application is currently in the **prototype stage**.

---

## 🧠 Key Features

### 🔬 Eye Disease Scan (AI Vision)

Users can take or upload an eye image to:

- Detect possible eye diseases using AI
- Automatically crop eye region using face detection
- View confidence scores
- Receive basic medical recommendations

Supported screening categories:

- Cataract
- Conjunctivitis
- Uveitis
- Eyelid Disease
- Normal

---

### 💬 Symptom-Based Diagnosis (AI Text)

Users can describe eye-related symptoms in natural language.

Example:

> "I have red eyes, irritation, and tearing."

The AI system will:

- Estimate probability of possible conditions
- Provide preliminary medical guidance
- Support Thai language input

---

## 🏥 Medical Purpose

Eye Protecs is designed for:

- Early risk awareness
- Health screening assistance
- AI medical research
- Educational demonstration

⚠️ This is NOT a medical diagnosis tool.

Users must consult a licensed ophthalmologist for professional diagnosis.
